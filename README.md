# Expedia-Clone
A website to book airline tickets, hotel reservations, car rentals, cruise ships, and vacation packages.

## Tech Stack
 react | react-router-dom | CSS

 ## Features

 - Sign Up
 
 - Sign In
 
 - Authentication
 
 - Search Hotel

 - Booking Hotel
